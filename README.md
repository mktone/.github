## 💡 서비스 소개

뉴스 기사를 Claude AI로 수어 문체에 맞게 변환하고, 3D 아바타가 실시간으로 수어를 재생하는 서비스입니다.

청각장애인이 뉴스를 수어로 자연스럽게 접할 수 있도록 텍스트 → 수어 키포인트 → 아바타 모션까지 전 과정을 자동화했습니다.
